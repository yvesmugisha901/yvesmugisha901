## 🔥 Featured Projects

### 🛒 Fasha Market (E-commerce Platform)
A secure e-commerce platform solving the problem of untrusted online marketplaces by enforcing verified sellers, agreements, and buyer protection.

🔗 Live Demo: http://fashamarket.vercel.app/

**Key Features:**
- Verified sellers with admin approval system  
- Buyer protection & return guarantee  
- Order tracking until delivery  
- Secure authentication & transactions  

**Tech Stack:**
- Frontend: React, TypeScript  
- Backend: Node.js, Express  
- Security: JWT, Helmet.js  
- DevOps: Docker, CI/CD pipelines, Cron Jobs  
- Media: Cloudinary  
- SEO Optimization (robots.txt included)
